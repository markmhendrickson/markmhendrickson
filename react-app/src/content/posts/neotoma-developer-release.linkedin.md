My startup Neotoma is now available as a developer release.

I've been running workflows through AI agents for the past year: email, tasks, finance, contacts and much more. The agents are capable. The thing that keeps breaking is trust: memory changes implicitly, context drifts, and I can't trace a wrong answer back to its source or use the same records across tools.

Neotoma is a structured memory layer for agents. Typed entities, stable IDs, full provenance, deterministic queries. Your data stays on your machine. One graph works across ChatGPT, Claude, and Cursor via MCP.

This release is CLI + MCP + OpenAPI, local-only. I'd like your help hardening it. Run it, hit edge cases, and report bugs.
