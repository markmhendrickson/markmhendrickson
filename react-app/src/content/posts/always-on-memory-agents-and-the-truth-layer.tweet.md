Last week, Google PM @Saboo_Shubham_ open-sourced an "always-on memory agent" that ditches vector databases entirely. LLM reads, thinks and consolidates memory without any embeddings.

It validates that need for persistent agent memory with more reliability than pure semantic search, while making opposite choices from a truth layer on every axis: automatic vs explicit, probabilistic vs deterministic, mutable vs immutable.

The strongest agentic architecture may contain both: consolidation agents that think, but running on top of stateful layers you can trust.
